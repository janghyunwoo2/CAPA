⚡ Before vs After

항목 Before After
데이터 조회 데이터팀 요청 → 3시간~3일 대기 @capa-bot → 1분 내 답변
주간 리포트 데이터 수집 + 정리 + 작성 = 4시간 "리포트 만들어줘" → 1분 내 생성
이상 감지 다음날 리포트에서 뒤늦게 인지 이상 패턴 감지 → 즉시 알림
KPI 확인 SQL 몰라서 데이터팀 요청 Redash 대시보드에서 직접 확인

🛠️ 기술 스택

📦 Data Pipeline

  • Kinesis Data Stream
  • Kinesis Firehose
  • S3 (Parquet)
  • Glue Catalog
  • Athena

🤖 AI/ML

  • Vanna AI (Text-to-SQL)
  • ChromaDB (Vector DB)
  • OpenAI GPT-4 / Claude
  • Lambda (이상 탐지)

🏗️ Infrastructure

  • Terraform (IaC)
  • Amazon EKS
  • Apache Airflow

👥 Interface

  • Slack Bolt
  • FastAPI
  • Redash
