
만약 두번째 질문에 첫번째 질문과 관련없는거 넣으면
 어떻게 동작하나?

정상적으로 동작합니다.

  1번째: "어제 기기별 클릭수"
  2번째: "이번달 캠페인별 ROAS 알려줘"
    → Step 0: history=1건 (이전 이력 있음)
    → Step 2 정제: "이번달 캠페인별 ROAS(광고
  수익률)"  ← 이전 맥락 무시하고 독립 질문 처리
    → Redash: 5건 반환 ✅

  이전 이력이 있어도 관련 없는 질문은 정상적으로 독립
   처리됩니다. QuestionRefiner가 이전 대화와
  연결고리가 없으면 그냥 현재 질문만 정제하고, 이전
  session_id, turn_number만 올라갑니다.

  즉 현재 동작:

  ┌──────────────────┬───────────────────────────┐
  │       경우       │   QuestionRefiner 동작    │
  ├──────────────────┼───────────────────────────┤
  │ "그 중에서",     │ 이전 맥락 참조 → 구체적   │
  │ "두번째" 등      │ 조회 조건으로 변환        │
  │ 지시어           │                           │
  ├──────────────────┼───────────────────────────┤
  │ 완전히 새로운    │ 이전 맥락 무시 → 독립     │
  │ 질문             │ 질문으로 정제             │
  └──────────────────┴──────────────────────────

왜 바차트만 나오나

  AI Analyzer 프롬프트 기준:
  "bar":  categorical comparisons (campaign/platform
  ranking)  ← 현재 질문들이 전부 여기 해당
  "line": time series (daily/weekly/monthly trend)

  지금까지 한 질문들:

- "어제 기기별 클릭수" → 기기 비교 → bar
- "캠페인별 CTR" → 캠페인 비교 → bar
- "지역별 노출수 순위" → 지역 비교 → bar

  모두 카테고리 비교 형태라 bar가 맞습니다.
