## CAPA 프로젝트 타겟 사용자 (프로젝트 컨셉 v4 기준)

> **"광고 비즈니스에서 데이터를 다루는 모든 비개발자"**

| 타겟 | 역할 | 데이터 니즈 |
|------|------|-----------|
| **AD Ops (광고 운영)** | 캠페인 세팅/모니터링/최적화 | 실시간 성과, 이상 징후, 일별 추이 |
| **PM / 기획자** | 서비스 기획, 성과 리포트 | 전체 트렌드, KPI 달성률, 비교 분석 |
| **세일즈** | 광고주 영업, 성과 보고 | 광고주별 성과, 매출, ROI |
| **퍼포먼스 마케터** | 광고 효율 분석, 예산 배분 | CTR/ROAS 분석, 소재별 성과, A/B 비교 |

## 타겟별 예상 자연어 질문 Top 10

### 1. AD Ops (광고 운영)

일상적으로 캠페인을 모니터링하고 이상을 빠르게 감지해야 하는 역할.

| # | 예상 질문 |
|---|---------|
| 1 | "어제 전체 캠페인 노출수, 클릭수, CTR 알려줘" |
| 2 | "오늘 클릭수가 0인 활성 캠페인 목록 보여줘" |
| 3 | "지난 7일간 CTR이 가장 많이 떨어진 캠페인 5개" |
| 4 | "어제 bid_price 합계가 일일 예산 대비 몇 % 소진됐는지 캠페인별로" |
| 5 | "모바일 vs 데스크탑 노출 비율 추이 (최근 7일)" |
| 6 | "어제 전환 이벤트가 발생한 캠페인 목록과 전환수" |
| 7 | "지난주 대비 이번주 전체 클릭수 변화율" |
| 8 | "특정 캠페인(camp_5)의 최근 30일 일별 노출/클릭 추이" |
| 9 | "어제 시간대별 트래픽 분포 보여줘" |
| 10 | "전환율(CVR)이 급락한 캠페인 있으면 알려줘 (전일 대비 50% 이상 하락)" |

| # | 예상 질문 |
|---|---------|
| 1 | "모바일에서 CTR이 가장 높은 캠페인 TOP 5" |
| 2 | "시간대별 클릭률 분포 (어떤 시간에 클릭이 가장 많은지)" |
| 3 | "디바이스별 ROAS 비교 (모바일 vs 데스크탑 vs 태블릿)" |

## 질문 패턴 분류

위 40개 질문을 패턴별로 분류하면 다음과 같습니다:

| 패턴 | 예시 | 비율 |
|------|------|------|
| **기간별 성과 조회** | "어제 CTR", "이번 달 매출" | 30% |

| **추이/트렌드** | "일별 CTR 추이", "월별 매출 트렌드" | 10% |

## Few-shot SQL 우선 구축 대상

위 패턴 분석을 기반으로, ChromaDB에 먼저 학습시켜야 할 예제 SQL 우선순위:

| 순위 | 패턴 | 이유 |
|:----:|------|------|
| 1 | **기간별 성과 조회** | 모든 타겟이 가장 많이 하는 질문. 파티션 조건 필수 |
| 2 | **순위/TOP N** | ORDER BY + LIMIT 패턴. 간단하지만 빈도 높음 |
| 3 | **비교 분석** | GROUP BY + CASE WHEN 패턴. 디바이스/기간 비교 |
| 4 | **이상/알림** | LAG 윈도 함수 등 고급 패턴. 정확한 예제가 없으면 실패율 높음 |
| 5 | **추이/트렌드** | 날짜별 집계. 시각화 연결 시 자주 사용 |
